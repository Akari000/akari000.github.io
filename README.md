# Setting
```
$ gem install jekyll
$ bundle install
```

# Build
Make sure to run if you change the yaml or git push.
```
$ jekyll build
```

# Watch on localhost
```
$ bundle exec jekyll serve
```
Then, you can see this site here: http://127.0.0.1:4000/
